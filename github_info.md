# GitHub Repository Information

**Repository Name:** 
`MRI-Bone-3D-Reconstruction-Pipeline`

**Description:**
High-fidelity 3D surface reconstruction pipeline for MRI bone segments (Radius & Ulna). Converts pre-segmented volumetric masks into accurate, watertight (manifold) STL models using Gaussian volume smoothing, Marching Cubes, and non-shrinking Taubin refinement.
